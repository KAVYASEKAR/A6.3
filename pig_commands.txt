Explain the working of below pig script commands with an example:

1. LOAD : A LOAD statement reads data from the file system.

2. STORE : A STORE statement writes output to the file system

3. DUMP : A DUMP statement displays output to the screen.

4. FOREACH : FOREACH operator is used to generate specified data transformations based on the column data.

5. FILTER : The FILTER operator is used to select the required tuples from a relation based on a condition.

6. GROUP BY : The GROUP BY operator is used to group the data in one or more relations. It collects the data having the same key.

7. ORDER BY : The ORDER BY operator is used to display the contents of a relation in a sorted order based on one or more fields.

8. DESCRIBE : DESCRIBE operator reviews the schema of a relation.
